API doc

The following API endpoints are accessible through this REST API

GET

- all stocks: http://localhost:8080/api/stocks
- stock with id: http://localhost:8080/api/stocks/{id}
- all cameras: http://localhost:8080/api/cameramodel
- camera with id: http://localhost:8080/api/cameramodel/{id}
- insert new roll of film: http://localhost:8080/api/roll?cameraid={id}&stockid={id}
- all rolls of film: http://localhost:8080/api/roll
- roll of film with id: http://localhost:8080/api/roll/{id}
- all rolls of film in development: http://localhost:8080/api/roll/in-development

POST

- create new stock: http://localhost:8080/api/stocks
- create new camera: http://localhost:8080/api/cameramodel

PUT

- give roll to development: http://localhost:8080/api/roll/{id} (id=Camera ID to empty roll out of)

DELETE

- remove roll from camera: http://localhost:8080/api/roll/{id} (id=cameraId to remove from)
